# Nudge — спецификация стиля и поведения

Описание на уровне ощущений и функциональности, а не попиксельной вёрстки. Цель — чтобы человек или модель могли воссоздать то же самое по удобству и внешнему виду в любой реализации (native Rust, Electron, WASM).

## 1. Окно и фон

- Без системного заголовка и рамки.
- Всегда поверх других окон, пока видно.
- Нельзя ресайзить.
- Не появляется в панели задач и в ALT+TAB.
- Живёт в трее: закрытие попапа не завершает приложение.
- Окно прозрачное везде (native и WASM). В WASM за карточкой рендерится статичная картинка-обои, чтобы прозрачность и тень карточки были видны при верификации в браузере.
- Палитра: только оттенки серого, без акцентного цвета.
- Позиция карточки: горизонтально по центру экрана, вертикально на ~40% высоты от верха (чуть выше середины).

## 2. Карточка

- Поверхность: полупрозрачная с блюром (frosted glass). Сквозь карточку размыто просвечивает то, что за окном — рабочий стол или другие окна.
- Ширина около 480px, высота растёт от контента.
- Скругление углов заметное и мягкое.
- Тень: минимальная по базе; допустимо чуть мягче для эстетики.
- Три строки одинаковой высоты, разделённые тонкой горизонтальной линией. Между крайними строками и краями карточки — такой же отступ, что и между полями.

## 3. Поля ввода

- Лейблы: placeholder внутри поля, исчезает при вводе. Отдельных подписей сверху/сбоку нет.
- Тексты placeholders:
  - Поле 1: `Что я делаю?`
  - Поле 2: `Хуйня?`
  - Поле 3: `Следующий через (мин)`
- Фокус на первом поле при открытии.
- Навигация: Tab вперёд, Shift+Tab назад.
- Активная (с фокусом) строка подсвечена лёгким изменением фона. Никаких рамок, outline-ов или цветных акцентов.
- Поле минут принимает целые и дробные числа (напр. `0.1` = 6 секунд, для тестов).
- Состояние полей при закрытии:
  - `doing` и `bullshit` очищаются при любом закрытии попапа.
  - Минутное поле сохраняет последний введённый текст, даже если пользователь не нажал Enter. В следующий раз попап открывается с этим значением.

## 4. Поведение

- **Enter**: записать в журнал → скрыть попап → перезапустить таймер с текущим значением поля минут.
- **Esc**: скрыть попап → перезапустить таймер с текущим значением поля минут. Без записи.
- **Клик мимо карточки** (за пределами её контура): равнозначно Esc.
- **Потеря фокуса окна** (пользователь кликнул в другое окно): равнозначно Esc.
- **Клик на трее / pill**: открыть попап принудительно (даже если таймер ещё не истёк).
- Появление и скрытие попапа — мгновенные, без анимаций.
- Первый запуск приложения сразу показывает попап. Дальше — скрытие → таймер → попап → ...

## 5. Трей (native)

- Иконка: стилизованный таймер в духе Gestimer — круглый циферблат с визуальным индикатором оставшегося времени. Monochrome, белая/светло-серая для тёмного таскбара Windows.
- Tooltip: `Nudge in N min`, обновляется раз в минуту (без секунд).
- Левый клик по иконке: показать попап.
- Правый клик: контекстное меню с двумя пунктами: `Show Nudge`, `Quit`.
- Язык трея (tooltip и меню): английский. Карточка остаётся на русском.

## 6. Pill (эмуляция трея в WASM)

- Рендерится только в браузере, только когда попап скрыт.
- Позиция: правый нижний угол, отступ ~16px от краёв.
- Форма: вытянутый pill-овал со скруглением до полного.
- Фон: плоский тёмно-серый, темнее карточки. Без frosted glass.
- Содержимое: только время, формат `M:SS` (минуты:секунды), без префикса. Обновляется раз в секунду.
- Hover: фон чуть светлее.
- Клик: открыть попап.

## 7. Журнал

Визуально не проявляется, но задаёт сценарий ошибок:

- Append-only NDJSON (одна JSON-запись на строку). Полный контракт — в `docs/journal-spec.md`.
- При ошибке записи журнала (неверный формат, нет прав) — показать системный диалог, попап остаётся открытым, введённый текст не теряется.

## 8. Открытые задачи на реализацию

Не визуальные решения, но вытекают из спеки:

- Native Windows: прозрачное окно + frosted-glass backdrop за карточкой (проверить поддержку в eframe / через Windows acrylic/mica).
- Native Windows: убрать из панели задач и из ALT+TAB.
- Native Windows: обработчик потери фокуса → dismiss + сброс таймера.
- Native Rust: tooltip трея обновляется раз в минуту.
- Иконка трея: SVG/PNG в стиле Gestimer (circular timer), monochrome, нужен дизайн.
- WASM: статичная картинка-обои в `e2e/dist/`, рисуется за карточкой.
