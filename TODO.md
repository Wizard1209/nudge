## Окно настроек

Открывается из трея (правый клик → `Settings`) как отдельный процесс
`nudge.exe --settings`. Сейчас содержит три поля:

- [x] **Хоткей.** Кнопка «Запись» — пользователь нажимает желаемую комбо, мы её ловим (через `hotkey::hotkey_from_egui`), канонизируем и подставляем в поле. Save персистит как обычно. Bare Escape отменяет запись.
- [x] **Автозапуск с Windows.** Тумблер. Транзакционно: пишем в `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`, подтверждаем через `is_enabled`, и только тогда фиксируем в config.json. На ошибку — плашка, чекбокс не сдвигается.

- [x] **Live-reload основного процесса** при изменении config.json. Watcher через `notify` следит за `<Documents>/Nudge/config.json` и сигналит тред трея перечитать файл; единственный живой эффект — перерегистрация глобального хоткея, остальные поля обновляются в кэше без UI-действий.
